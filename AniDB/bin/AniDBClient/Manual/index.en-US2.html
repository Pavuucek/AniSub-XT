Description:

It looks like AOM or WebAOM, but it is fast as AME. After hashing your files, the application storage infos about them to local database. So you can browse your files in near future. You can search, browse anime or files or episodes, filter files and other more stuff like add/edit/delete entries in MyList and more.

And also application can renaming and moving files. Syntax of programing rules for renaming/moving is similar as syntax in WebAOM/AME (I hope it is similar )

Watch dog:

Enable and Adding directories:
1) Add directories to list
2) Enable files watcher
3) Restart AniSub

Remove diroctories:
1) Remove directories from list
2) Restart AniSub

Disable
1) Disable files watcher
2) Restart AniSub

Commandline option:

"Path_to_Anisub.exe Path_to_directory"
"C:/Program Files/Grep-a-Fruit/AniSub/AniSub.exe F:/Download/Anime"

Button parse in Hash Tab:

You can add in hash list this files: *.sfv, *.md5, *.ed2k and *.xml

*.sfv, *.md5, *.ed2k:
Parse data and update local file path in database.
Files which isn't stored in database will be added into hash list.
Parsing *.ed2k will skip hashing files which ins't stored in database.

How to add whole MyList:

This isn't TCP client for AniDB, so you can't download whole anime database from AniDB! And you can't download your mylist from AniDB! But there is one way, how to add your mylist from AniDB to AniSub.

1) You will have to do mylist export.
2) Select "html-fstyle" template.
3) Click on "Request export".
4) Wait till AniDB do export.
5) Download your export to hardisk.
6) Unzip/unrar export.
7) In AniSub (Index tab) add ".xml" extension to list.
8) Select all xml files in unziped folder and drag them all to AniSub.
9) Click on parse button (Hash tab).

Now AniSub will add files and anime(s) to your database. But you will have to do another export to add mylist stats to your database.

1) You will have to do new mylist export.
2) Select "txt-udp-mylist" template.
3) Click on "Request export".
4) Wait till AniDB do export.
5) Download your export to hardisk.
6) Unzip/unrar export.
7) In AniSub (Index tab) add ".txt" extension to list.
8) Select mylist.txt in unziped folder and drag it to AniSub.
9) Click on parse button (Hash tab).

Tags:
%ann - Anime name JAP
%kan - Anime name KAJ
%eng - Anime name ENG
%epn - Episode name ENG
%epk - Episode name KAJ
%epr - Episode name JAP
%enr - Episode number
%grp - Group name short
%ed2 - ed2k lower case
%ED2 - ED2K upper case
%md5 - md5 lower case
%MD5 - MD5 upper case
%sha - sha1 lower case
%SHA - SHA1 upper case
%crc - crc32 lower case
%CRC - CRC32 upper case
%ver - File version
%inv - CRC not match, etc.
%cen - Censored: cen, unc
%dub - Dub language
%sub - Sub language
%vid - Video codec
%aud - Audio codec
%qua - File quality
%src - File source
%res - Video resultion

%eps - Anime total episodes

%gen - Category string
%fid - File ID
%aid - Anime ID
%eid - Episode ID
%gid - Group ID
%ani - AniDB file name
